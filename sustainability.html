Sustainability

Overview

Create a dedicated landing page highlighting high-level content in the NRG 2015 Sustainability Report. Additionally, the page should be inspiring, engaging, efficient and thoughtful. The page should reflect the report and act as a "sneak peek" into the contents within the report. Include a CTA button where users will be able to directly download the report as well.

Screenshot of webpage. Live site can be found at www.nrg.com/sustainability

Objectives

  1. Make it easy for users to download the PDF
  2. Make the content accessible for all audience segments. The content should reflect the content within the 2015 Sustainability Report.

Research

I created multiple personas based of off target audiences. These personas helped hone in and identify users’ end goals. A total of six personas were created and six user journeys (one per persona) were created as well. With this research, I provided UX recommendations and a sitemap for the landing page for the 2015 Sustainability Report, as well as the sustainability section. I also provided visual examples of similar sites to use as reference for brainstorming look and feel.

2016 NRG Sustainability Report where users could download the sustainability report, view and download visually appealing charts and graphs and act as a “sneak preview” of the contents found within the sustainability report.
